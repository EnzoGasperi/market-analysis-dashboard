# Market Analysis Dashboard

Automated multi-asset trading analysis system built in Python. Generates a dynamic HTML dashboard with real-time financial data, technical indicators, and risk management parameters.

## What it does

- Fetches real-time price data for Bitcoin, Gold, and NASDAQ Futures via `yfinance`
- Calculates 8+ technical indicators: RSI, MACD, ATR, ADX, Bollinger Bands, StochRSI
- Integrates economic calendar events and filters macro news relevant to each asset
- Applies automated risk management rules (daily stop-loss, daily profit target)
- Generates a fully interactive HTML report with trade signals and market context

## Tech stack

- Python 3.x
- pandas, numpy, yfinance, pandas-ta, pytz
- HTML report generation (no external dashboard server required)
